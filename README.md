# 🚀 AI Resume Ranker

> An intelligent Resume Ranking System built using **Python, Flask, and NLP** to help recruiters efficiently shortlist candidates based on job descriptions and skill matching.

---

## 📌 Overview

AI Resume Ranker is a smart web application that analyzes uploaded resumes and ranks them according to their relevance to a given job description using Natural Language Processing techniques.

This project helps automate the initial resume screening process and reduces manual effort in recruitment.

---

## ✨ Features

✅ Upload Multiple Resumes
✅ Supports PDF & DOCX Resume Formats
✅ Extracts Resume Text Automatically
✅ NLP-Based Resume Analysis
✅ Resume Ranking Based on Similarity Score
✅ Simple & Interactive User Interface
✅ Fast and Efficient Screening Process

## ⚙️ Installation & Setup

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Yesaswini05/resume-ranker.git
```

### 2️⃣ Navigate to Project Folder

```bash
cd resume-ranker
```

### 3️⃣ Install Required Packages

```bash
pip install -r requirements.txt
```

### 4️⃣ Run the Application

```bash
python app.py
```
